# ⚛️ What is React JS?

## 🔹 Overview

**React JS** is an open-source **JavaScript library for building user interfaces**, especially for **single-page applications (SPAs)**.

It was developed by **Facebook (Meta)** and allows developers to create reusable UI components that efficiently update when data changes.

React uses a **component-based architecture** that makes development faster, scalable, and easier to maintain.

---

## 🔹 Key Features of React

- Component-Based Architecture
- Virtual DOM for fast rendering
- Reusable UI Components
- One-way Data Binding
- Strong Community Support
- Efficient state management using Hooks
- Easy integration with APIs and backend systems

---

## 🔹 Why Use React?

- Faster UI updates using **Virtual DOM**
- Clean and reusable component structure
- Large ecosystem and community
- Used by major companies like **Facebook, Netflix, Instagram, and Airbnb**
- Excellent for building **modern web applications**

# 🧠 React Concepts Covered

This repository includes practice with the following concepts:

### 1️⃣ Components

Reusable UI building blocks.

```jsx
function Navbar() {
  return <h1>Welcome to React</h1>
}
```

---

### 2️⃣ Props

Used to pass data between components.

```jsx
function Greeting(props) {
  return <h1>Hello {props.name}</h1>
}
```

---

### 3️⃣ useState Hook

Used for managing component state.

```jsx
import { useState } from "react"

const [count, setCount] = useState(0)
```

---

### 4️⃣ useEffect Hook

Used for handling side effects like API calls.

```jsx
useEffect(() => {
  console.log("Component mounted")
}, [])
```

---

### 5️⃣ Event Handling

```jsx
<button onClick={() => setCount(count + 1)}>
  Click Me
</button>
```
